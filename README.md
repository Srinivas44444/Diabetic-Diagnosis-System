# A DATA CENTRIC APPROACH TO IMPROVE DIABETES DIAGNOSIS ACCURACY
# Project Description
Developed an AI-based Diabetes Diagnosis System integrating Machine Learning (Gradient Boosting Classifier) and Deep Learning (CNN architectures like ManualNet and LeNet-5) for accurate diabetic prediction and retinal image classification. Deployed using Django for real-time medical diagnosis.
# Implementation Methods
* Data Acquisition and Pre-processing
* Exploratory Data Analysis and Visualization
* Machine Learning Model Development
* CNN-based analysis for Diabetic Retinopathy
* Django Framework Integration and Deployment
# Algorithms and Architecture
* Gradient Boosting
* Manual Net Architecture
* Lent Architecture
# Features
* Hybrid Diagnostic Approach
* Automated Medical Image Analysis
* Clinical Data Processing
* User friendly Web Interface for Healthcare Access
* Early Detection of Complications
# Tech Stack
* Languages: Python, HTML, CSS, JavaScript
* Frameworks: Django, Tensorflow, Keras
* Libraries: Pandas, Numpy, Matplotlib, Scikit-learn
* Deployment: Django, SQLite
# Usage
* Upload Retinal images for diabetic retinopathy detection.
* Enter medical details to predict diabetes risk.
# Results
* Gradient Boosting Classifier achieved optimal performance on the dataset after data preprocessing and feature selection.
* The LeNet model reached 97% accuracy after training for 200 epochs.
